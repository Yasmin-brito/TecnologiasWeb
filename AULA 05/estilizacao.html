Exemplos de Estilização de Texto com CSS

font-size: Este texto tem um tamanho de fonte maior usando font-size: 24px

font-weight: Este texto está em negrito com font-weight: bold.

font-style: Este texto está em itálico com font-style: italic.

text-align: center. Este paragrafo está centralizado horizontalmente

text-align: right Este parágrafo está alinhado à direita.

line-height: Esta frase tem espaçamento de linha aumentado para facilitar a leitura.

O valor aplicado é line-height: 2 (2 vezes o font-size)

letter-spacing: espaçamento entre letras usando letter-spacing: 4px.

word-spacing: Este texto tem espaçamento maior entre palavras usando word-spacing: 20px.

text-shadow: Texto com sombra aplicada usando text-shadow: 2px 2px 4px.
